## 超级逗表情 App
###  需求
    * 选用ionic前端框架
    * 选用cordova作为打包工具
    * 支持iOS 和 android平台
    * 支持分享到qq，微信，新浪

### 项目地址
    * [https://github.com/apanly/chaojidbq](https://github.com/apanly/chaojidbq)
    
### 项目开发步骤
    * ionic start chaojidbq blank
    * ionic platform add ios
    * ionic platform add android
